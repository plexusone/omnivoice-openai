## [v0.2.0] - 2026-04-19

### Highlights

- Module renamed from omnivoice-openai to omni-openai
- Added OmniLLM provider adapter for chat completions

### Breaking

- Module path changed from github.com/plexusone/omnivoice-openai to github.com/plexusone/omni-openai ([`aa9d5da`](https://github.com/plexusone/omni-openai/commit/aa9d5da))

### Added

- OmniLLM provider implementing core.Provider interface with streaming, tool calling, vision, and JSON mode support ([`090462b`](https://github.com/plexusone/omni-openai/commit/090462b))
- Auto-registration with omnillm-core registry at thick provider priority ([`090462b`](https://github.com/plexusone/omni-openai/commit/090462b))
- MkDocs documentation site with getting started guide, provider docs, and API reference

## [v0.1.0] - 2026-02-28

### Highlights

- Initial release of omnivoice-openai

### Added

- OmniVoice STT provider implementing `stt.Provider` interface ([`ee8c5f8`](https://github.com/plexusone/omni-openai/commit/ee8c5f8))
